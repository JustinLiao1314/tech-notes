### 索引

索引就是用来加速查询的。`MongoDB` 的索引与其他关系型数据库索引几乎一模一样。所以其他关系型数据库的索引技巧一样可以用于 MongoDB 。

#### 创建索引

```js
db.blog.ensureIndex({ "username": 1 }, options);
```
